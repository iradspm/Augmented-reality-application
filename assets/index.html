User guide to augmented reality

The Image Targets provided in the table below should be downloaded so that one can be able to scan those targets and be able to interact with augmented contents.

Key Functionality

  • Getting an input using your phone camera
  • Augmenting the target image
  • Interacting with augmented content: For 3D augments, you can interact with it by rotating image targets
  • Teaching guide through video clip: On successful scan of image targets, a video explaining drawing conceipt is shown guiding the students on how to draw.

Instructions

  • Point camera at target to scan it
    This ensures necessary features of image targets are recognized well.

Softwares used

  • Vuforia sdk
  • Android studio IDE
  • Unity IDE
